# VuePress Github Pages

Use VuePress to built a static website and deploy on Github Pages.

# Demo

See this page on https://wei-xia.github.io/VuePress-template/

# Start

Make sure that you have NodeJS and NPM or Yarn installed.

# Install

npm install

or

yarn

# Develop

npm run start

or

yarn start

# Build

npm run build

or

yarn build

Warning

Make sure node version >= 8.0
